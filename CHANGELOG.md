## [0.0.1] - 2025-02-13

- Initial release
- Script bar panel in bottom panel with buttons for package.json scripts
- Enable/disable scripts, nicknames, and custom button colors via `.vscode/package-script-bar.json`
- Run scripts in background (with completion notifications) or in terminal
- Rescan package.json to merge new scripts without overwriting preferences
- Team-shareable config file
